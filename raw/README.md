# Raw Sources

Hier werden Quellen abgelegt, die vom LLM verarbeitet werden sollen.
Dateien in diesem Ordner werden **nie verändert** — sie sind die Source of Truth.

Beispiele:
- Papers (PDF): `chronos-2-paper.pdf`
- Artikel (Markdown): `nixtla-nhits-blog.md`
- Dokumentation: `neuralforecast-docs.md`
- Meeting Notes: `2026-04-14-forecast-review.md`
- Experiment Logs: `experiment-log-nhits-v2.md`

**Workflow**: Datei hier ablegen → Claude Code sagen "Ingest raw/dateiname"
→ Wiki wird automatisch aktualisiert.
